# Discovery

Discovery covers the initial product decisions for Docs Viewer: source format, runtime shape, publishing modes, and the decision to keep markdown as the source of truth.

## Outcome

- Use a markdown vault as input.
- Build a static JSON index.
- Keep the browser runtime dependency-light.
- Add Node only for auth, config, and rebuild endpoints.

# Indexing

Indexing turns the source vault into `viewer/data/vault-index.json`.

## Scope

- Parse YAML frontmatter.
- Resolve aliases and wikilinks.
- Build backlinks.
- Copy supported image assets.
- Generate a folder tree for sidebar navigation.
